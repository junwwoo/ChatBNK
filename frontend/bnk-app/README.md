# 🚀 Create React App 프로젝트

이 프로젝트는 **Create React App**으로 생성된 React 애플리케이션입니다.

---

## 📦 실행 방법

프로젝트 디렉토리에서 아래 명령어를 실행하세요.

### ▶️ 개발 서버 실행

```bash
npm start
```

- 개발 모드로 앱이 실행됩니다.
- 브라우저에서 http://localhost:3000 접속
- 코드 수정 시 자동 리로드
- 콘솔에서 에러 및 경고 확인 가능

---

### 🧪 테스트 실행

```bash
npm test
```

- 테스트를 watch 모드로 실행합니다.
- 파일 변경 시 자동으로 테스트 재실행

---

### 🏗️ 프로덕션 빌드

```bash
npm run build
```

- `build/` 폴더에 최적화된 결과물 생성
- 코드 압축 및 성능 최적화 적용
- 파일명에 해시 포함 (캐싱 대응)

👉 배포 가능한 상태의 정적 파일이 생성됩니다.

---

### ⚙️ 설정 커스터마이징 (Eject)

```bash
npm run eject
```

⚠️ **주의: 되돌릴 수 없습니다**

- webpack, Babel, ESLint 설정이 프로젝트 내부로 복사됩니다.
- 이후 모든 설정을 직접 수정할 수 있습니다.

💡 일반적인 경우 eject 없이도 충분히 개발 가능합니다.
